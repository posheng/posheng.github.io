Common commands :

  • aws ec2 run-instances: This command launches the specified number of instances using an AMI for which you have permissions.
  • aws ec2 describe-instances: Describes one or more of you instances including InstanceId, State, Name, etc.
  • aws ec2 create-volume: Creates an Amazon EBS volume that can be attached to an instance in the same Availability Zone.
  • aws ec2 create-vpc: Creates a VPC with the specified CIDR block. • aws s3 ls: Lists Amazon S3 objects and common prefixes under a prefix or all Amazon S3 buckets.
  • aws s3 cp: Copies a local file or Amazon S3 object to another location locally or in Amazon S3.
  • aws s3 mv: Moves a local file or Amazon S3 object to another location locally or in Amazon S3.
  • aws s3 rm: Deletes an Amazon S3 object.
